Manual

Functionality

This application lets you print simple labels or let you create labels of csv files with profiles.

-Print simple labels

Go to tab "Label" and enter the text you want print later. To print press the "print" button. A print dialog is opening now.

-Print from csv files

First you have to open a csv file under File->Open. Now the content of the csv file is loaded into the application and show the data in a table. You can now choose one of your definded profiles from a combo box and press "print" to print the data with the settings of the profile.

-The print dialog

When clicking on the print button a applications print dialog open. Here you the a preview of the data which get printed when pressing Print button instead of printing you can save the content as html file by clicking the save button. When you want to ignore the printer size configurated in your printers driver you can check the "Change Size" checkbox and define a custom width and height.

Profiles

Here you can create settings for specific use cases.

-Name

The name of the template

-Seperator

The seperator which is used in the csv file to seperate data.

-Codec

The codec which is used to encode the csv file. Should be the same as the codec used in the csv file.

-Contains header

If the csv file use a header you should check this box, so the first row is not interpreted as data.

-Template

Here you can create a layout in html for your label. You can find supported html and css properties under: http://doc.qt.io/qt-5.8/richtext-html-subset.html. Also you can set placeholders which has the form $(i) where i is the column of your csv file. The first column has the number 1.

Settings

-File open path

The default start path when opening a file

-Custom Size defaults

The default with and size when selecting "Custom Size" in applications print dialog
